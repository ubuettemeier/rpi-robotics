The software for I2C handling is developed on the Raspberry Pi 3 B.
The programs work with the I2C kernel modules i2c_dev and i2c_bcm.

script's
- source/edit.sh => opens the source files with geany
- source/run.sh => starts the program build/test_laser_sensor
